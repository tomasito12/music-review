# Plattenradar Frontend

First React/Vite shell for the future Plattenradar web frontend.

## Commands

```bash
pnpm install
pnpm dev
pnpm build
pnpm test
```

The dev server defaults to `http://127.0.0.1:5173`.

## Current Scope

This is a shell MVP, not a full Streamlit replacement yet. It includes:

- global app shell and navigation,
- welcome screen,
- mocked `Aktuell` and `Entdecken` result screens,
- editorial recommendation cards,
- music profile setup shell,
- playlist generator shell,
- auth dialog shell,
- typed API client foundation.

The shell is intentionally designed to work before the full data-backed frontend
is connected.
